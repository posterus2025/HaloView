# HaloView Tools

Build scripts and benchmarking utilities.

## Contents
```
benchmarks/
  wifi_throughput.py    iperf3 wrapper for Wi-Fi 6E testing
  encode_latency.py     AMF encode latency measurement
  e2e_latency.py        End-to-end motion-to-photon measurement
scripts/
  setup_env.ps1         Dev environment setup (checks prerequisites)
  unload_llm.ps1        Safely unload Proculus LLM before VR session
  reload_llm.ps1        Reload Proculus LLM after VR session
```
